# Personal Portal - 个人工具站

一个轻量、优雅的个人效率工具集合，目前已集成多个模块，所有数据完全在本地浏览器存储，无后端服务器。

## 它能做什么

### 1. 个人门户

侧边栏导航 + 门户主页，统一入口跳转到各模块。

### 2. 实时工资计算

设定月薪和工作时间后，打开页面就能看到**精确到秒**的今日已赚金额，数字实时跳动更新。同时展示：

- 当前发薪周期（本月/跨月）已赚多少
- 本月预计到手多少钱
- 年度累计已赚、剩余工作日一目了然

### 3. 工作日历

文字标注（班 / 假 / 休 / 节）显示每一天的状态，内置 2024-2028 年中国法定节假日和调休数据，自动判断今天是否计薪。

### 4. 请假管理

在日历上点选请假日期（年假 / 事假 / 病假），系统自动扣除对应薪资，扣薪金额在薪资漏斗中可视化展示。

### 5. 简历储存

按 7 类信息（个人信息、教育、工作、项目、技能、证书、其他）以时间线方式收纳你的全部经历，浏览器本地持久化。

### 6. 定制简历（建设中）

三步流程：输入目标公司 + JD → 系统推荐相关条目 → 实时预览最终简历。当前 UI 框架与假数据联动已完成，真实数据接入、关键词匹配评分、AI 增强等能力开发中。

### 7. 主题系统

支持暗色 / 亮色主题切换，全屏沉浸体验，桌面端与移动端响应式布局。

### 8. 数据完全在本地

所有配置、请假记录、简历内容都存在浏览器 localStorage，无后端服务器，隐私安全。

## 适合谁用

- 按月薪计薪、想知道"我这一天值多少钱"的上班族
- 需要管理请假扣薪的职场人
- 在准备求职、想要按岗位定制简历的人
- 喜欢极简、纯本地、可离线的个人效率工具用户
