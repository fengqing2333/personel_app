# [V1.0.0] 实时工资计算器

## Why

作为按月薪计薪的实习生，想知道"我今天值多少钱"——希望打开网页就能看到实时跳动到手的金额，精确到秒。同时需要管理请假扣薪、追踪月度/年度薪资进度。全本地运行，无需后端。

## Goals

- 实时显示今日已赚金额，数字每秒跳动更新
- 展示本月/年度薪资进度追踪
- 支持请假管理（年假/事假/病假）自动扣薪
- 内置中国法定节假日和调休数据，自动识别工作日
- 暗色+亮色主题切换，全屏沉浸模式
- 纯本地运行，localStorage 存储

## Non-goals

- **不做税后计算**（实习生暂不涉及个税）
- **不做社保/公积金扣除**（实习生不涉及）
- **不做后端服务**（全部前端本地运行）
- **不做登录/用户系统**（单机单人使用）
- **不做多语言国际化**（仅中文）

## What Changes

- 新增实时工资计算引擎（秒薪计算、数字跳动）
- 新增工资看板页面（今日已赚、月度/年度进度）
- 新增工作日历组件（内置 2024-2028 中国节假日数据）
- 新增请假管理功能（年假/事假/病假、自动扣薪）
- 新增薪资设置模块（月薪/工时/计薪日配置）
- 新增全屏沉浸模式
- 新增暗色/亮色主题切换

## Capabilities

### New Capabilities

- `salary-dashboard`: 实时工资看板，包含今日已赚数字跳动、本月/年度薪资进度、周走势图
- `work-calendar`: 工作日历，内置中国节假日数据，标记工作日/休息日/请假日
- `leave-management`: 请假管理，支持年假/事假/病假添加，自动计算扣薪
- `salary-settings`: 薪资配置，可设置月薪、每日工时、月计薪天数
- `theme`: 主题系统，支持暗色/亮色切换

### Modified Capabilities

- 无（首个变更，尚无现有规范需要修改）

## Impact

- 新增前端页面和组件，无后端影响
- 新增中国节假日数据集（2024-2028 年）
- 新增 localStorage 数据持久化
- 技术栈待定（纯前端 SPA，Vite 构建）
