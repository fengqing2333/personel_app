# 实时工资计算器 — 设计文档

## 概述

为一个实习生场景设计实时工资计算网页，核心诉求是"打开页面就能看到今天赚了多少钱，数字实时跳动"。无社保/个税扣除，按 22 个工作日/月、固定 8 小时/日计算秒薪。

## 设计方向（已确认）

- **视觉风格**：暗色主题，青蓝色（#00c2ff → #0066ff）为主色调
- **布局**：左右双栏 — 左看板 + 右管理，桌面并排，移动端上下排列
- **核心元素**：实时跳动数字、本月/年度/目标三指标、周走势图、工作日历、请假管理、快捷设置

## 核心功能模块

### 1. 实时工资计算引擎
- 输入：月薪、每日工时、月计薪天数（22）
- 计算链：月薪 ÷ 22 ÷ 日工时 ÷ 3600 = 秒薪
- 今日已赚 = 秒薪 × 已过秒数（当日 00:00:00 起算）
- 每秒更新 DOM 中的数字显示

### 2. 薪资进度追踪
- 本月已赚：秒薪 × 本月已过工作秒数
- 年度累计：秒薪 × 今年已过工作秒数
- 进度条：本月已赚 / 月薪

### 3. 工作日历
- 内置 2024-2028 中国法定节假日和调休数据
- 自动标记工作日/休息日
- 点击日期可添加请假标记

### 4. 请假管理
- 三种类型：年假、事假、病假
- 请假天数 × 日薪（月薪 ÷ 22）= 扣薪金额
- 数据存储在 localStorage

### 5. 薪资设置
- 可配置：月薪、每日工时、计薪方式（22天/月）
- 修改后实时重算所有数据

### 6. 全屏沉浸模式
- 隐藏导航和其他面板，只显示核心数字
- 适合放在副屏或手机桌面

## 数据模型

```typescript
// 薪资配置
interface SalaryConfig {
  monthlySalary: number  // 月薪（元）
  dailyHours: number     // 每日工时
  workDaysPerMonth: number // 月计薪天数
}

// 请假记录
interface LeaveRecord {
  date: string           // YYYY-MM-DD
  type: 'annual' | 'personal' | 'sick'
}

// 节假日数据
interface Holiday {
  date: string           // YYYY-MM-DD
  type: 'holiday' | 'workday'  // 法定假或调休补班
  name: string
}

// 计算中间值
interface SalaryState {
  secondRate: number     // 秒薪
  todayEarned: number    // 今日已赚
  monthlyEarned: number  // 本月已赚
  yearlyEarned: number   // 年度累计
  leaveDeduction: number // 请假扣薪
}
```
