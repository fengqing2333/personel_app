# 个人门户主页 — 设计文档

## 概述
将现有工资计算页面整合为个人门户的一个功能模块，新增简历储存和每日一语。

## 布局
杂志风侧边栏：深紫渐变侧边栏(220px) + 奶油色主内容区

## 配色
- 侧边栏：深紫渐变 #1a0e2e → #2a1a3e
- 主背景：奶油色 #faf6f0
- 卡片：白色 + 24px 圆角 + 柔和阴影
- 文字：深棕 #3a2a1a

## 侧边栏
PORTAL 品牌标识 + 菜单项(概览/工资/简历)，14px 大圆角

## 主内容
每日一语横幅 + 两张大功能卡片(工资/简历)
