# 日历优化 — 设计文档

## 概述

优化工作日历组件的显示方式，用文字标注（班/假/休/节）替代纯颜色区分工作日、请假、节假日和休息日，提升可读性和清晰度。

## 设计方向（已确认）

文字标注版日历，每个日期下方显示类型文字：
- **班**（青蓝）— 工作日（含调休补班）
- **假**（粉色）— 已请假
- **休**（灰色）— 周末/非工作日
- **节**（金色）— 法定节假日
- **调**（小字青蓝）— 调休补班附加标签

## 布局变更

- 日历顶部：月份切换（左右箭头）+ 图例行（班/假/休/节文字图例）
- 日历网格：日期数字 + 下方文字标注
- 今日日期：蓝色渐变背景 + 加粗
- 底部：出勤/请假/休息/剩余 统计行
- 节假日：额外识别"节"标签
- 调休补班：在"班"下方附加"调休"小标签

## 字体大小

- 月份标题：18px 700
- 日期数字：15px（今日 17px 700）
- 文字标注：11px
- 调休小标签：9px
- 底部统计：13px

## 实现范围

- 重构 `WorkCalendar.vue` 组件模板和样式
- 更新 `AttendancePanel.vue` 中的数据统计联动
- 保持现有交互不变（点击添加/删除请假）
